# Git Tools

## Installation

Download the latest version and move it to your `$PATH`:

Using `wget`:

```bash
wget -O /usr/local/bin/git-push https://raw.githubusercontent.com/zharfanug/git-tools/latest/git-push.sh && chmod +x /usr/local/bin/git-push
wget -O /usr/local/bin/git-merge https://raw.githubusercontent.com/zharfanug/git-tools/latest/git-merge.sh && chmod +x /usr/local/bin/git-merge
```
Using `curl`:
```bash
curl -o /usr/local/bin/git-push https://raw.githubusercontent.com/zharfanug/git-tools/latest/git-push.sh && chmod +x /usr/local/bin/git-push
curl -o /usr/local/bin/git-merge https://raw.githubusercontent.com/zharfanug/git-tools/latest/git-merge.sh && chmod +x /usr/local/bin/git-merge
```
